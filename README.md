# NayePankh Foundation

NayePankh Foundation is a student-led NGO based in Kanpur, India. This website is the official digital portal for the organization, allowing volunteers to register, donors to contribute with 80G tax exemptions, and communities to track field impact.

## Features
- **Responsive Design**: fully optimized for desktop and mobile devices.
- **Premium Animations**: smooth scroll reveals, parallax effects, and intersection observers for a beautiful user experience.
- **Donation Wizard**: an intuitive, multi-step donation simulation showing real-time 80G tax exemption calculations.
- **Volunteer Portal**: accessible recruitment pipelines.
- **SEO & Accessibility**: Lighthouse-optimized with strict aria-labels, semantic HTML, and dynamic lazy-loading.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Custom Variables, Flexbox, CSS Grid)
- Vanilla JavaScript (Intersection Observers, DOM Manipulation)

## Deployment Guide

This website is completely static (HTML/CSS/JS) and does not require a build step or backend node process, making it perfectly compatible with instantaneous CDNs like Vercel, Netlify, or GitHub Pages.

### Deploying to Vercel
1. Push your repository to GitHub.
2. Log into [Vercel](https://vercel.com/) and click **Add New Project**.
3. Import this repository.
4. Leave the Framework Preset as **Other** and Build Command **empty**.
5. Click **Deploy**.

### Deploying to Netlify
1. Push your repository to GitHub.
2. Log into [Netlify](https://www.netlify.com/) and click **Add new site** > **Import an existing project**.
3. Connect your GitHub and select this repository.
4. Leave Build command empty and Publish directory empty (or set to `/`).
5. Click **Deploy site**.
